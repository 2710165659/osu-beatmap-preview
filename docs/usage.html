osu! Beatmap Preview

osu! 谱面预览工具 · 支持 Standard / Taiko / Catch / Mania 四模式 GIF 与 PNG 渲染 · 支持 Mod
单可执行文件 · 无运行时依赖 Windows / Linux / macOS 高性能

1 用法

osu-beatmap-preview --bid=<BID> [选项]

2 参数

--bid必填
Beatmap ID，程序唯一必填项。
--convert
转谱模式，可选 mania / ctb / taiko，仅 Standard 谱面可用。
--mods
Mod 组合，用 + 连接，如 hd+hr；倍速类可带数值如 dt1.25；DA 类可带参数如 daar5cs7。
--fmt
输出格式：gif 或 png，不填则按模式取默认值。
--time
GIF 时间点，最多四个，单位秒，用 + 连接，如 10+25+60。
--bpm
仅对 Taiko PNG 有效：指定排列间距 BPM，让 taiko note 按该 BPM 的节拍间距渲染，如 --bpm=180。
其他参数
--no-cache
跳过下载缓存与输出缓存，强制重新下载谱面并重新渲染。
--version
打印版本号与构建时间后退出，不执行渲染。

3 示例

基本
osu-beatmap-preview --bid=123456
转谱
osu-beatmap-preview --bid=123456 --convert=mania
强制GIF
osu-beatmap-preview --bid=123456 --convert=mania --fmt=gif
Mod组合
osu-beatmap-preview --bid=123456 --mods=hd+hr
时间点
osu-beatmap-preview --bid=123456 --time=10+25+60
复杂
osu-beatmap-preview --bid=123456 --convert=mania --fmt=gif --mods=4k+ds+in+dt1.2 --time=10+30+50

4 Mod 支持

Standard
GIF 支持
EZHR HDDA TCDTHT
PNG 支持
EZHR HDDATC
可转谱为 Taiko / Catch / Mania 预览（--convert）。
Taiko
GIF 支持
EZHR SWCS DTHT
PNG 支持
EZHRSW
默认 PNG 全谱面长图。
Catch
GIF 支持
EZHR DTHT
PNG 支持
EZHR
默认 PNG 全谱面长图。
Mania
GIF 支持
CSDTHT 1K–10KDSINHO
PNG 支持
1K–10KDSINHO
1K–10K 仅在 --convert=mania 时生效。默认 PNG 全谱面长图。
提示：粉色为 GIF 支持，蓝色为 PNG 支持。DT / HT / CS 仅 GIF 支持。

5 Mod 规则

DT/HT
互斥。DT 默认 1.5x，范围 1.01–2.00；HT 默认 0.75x，范围 0.50–0.99。
EZ/HR
互斥。
TC/HD
互斥。
1K–10K
互斥，仅 --convert=mania 时生效。
IN/HO
互斥。
DA/EZ/HR
DA 不能与 EZ 或 HR 同时使用，仅 Standard 可用。
DA 参数
格式 da<参数><值>，如 dacs5、daar9.5，可叠加如 dacs5ar9.5。
⚠ 冲突组合会直接报错，请按上述规则组合 Mod。
